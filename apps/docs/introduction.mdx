## What is Tutly?

Tutly is a comprehensive digital platform designed specifically for colleges and students. It brings together academics, collaboration, community engagement, and student portfolios into one seamless experience.

<CardGroup cols={2}>
  <Card title="Live Classes" icon="video" href="/features/live-classes">
    Built-in live class system with unlimited attendees and real-time engagement
    tracking
  </Card>
  <Card title="Assignments" icon="code" href="/features/assignments">
    Test case-based assignments with live playgrounds and auto-evaluation
  </Card>
  <Card title="Student Portfolios" icon="user" href="/features/portfolios">
    Unique public dashboards with personalized subdomains for every student
  </Card>
  <Card title="Events" icon="calendar" href="/features/events">
    Comprehensive event registration system with forms, certificates, and
    payment integration
  </Card>
</CardGroup>
