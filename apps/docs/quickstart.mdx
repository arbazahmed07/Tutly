## Getting Started with Tutly

Learn how to set up and start using Tutly for your institution.

### Setup Process

<AccordionGroup>
  <Accordion icon="building" title="For Institutions">
    To get started with Tutly for your institution: 1. Contact our team at
    support@tutly.in 2. Choose your deployment option (self-hosted or cloud) 3.
    Set up your institution's branding and customization 4. Configure your
    initial courses and user roles
  </Accordion>
  <Accordion icon="user" title="For Students">
    To access Tutly as a student: 1. Get your login credentials from your
    institution 2. Set up your student profile 3. Access your courses and
    assignments 4. Start building your portfolio
  </Accordion>
  <Accordion icon="chalkboard-user" title="For Teachers">
    To get started as a teacher: 1. Receive your teacher account credentials 2.
    Set up your teaching profile 3. Create your first course or class 4. Start
    scheduling live sessions
  </Accordion>
</AccordionGroup>
